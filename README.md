## 项目介绍
一个简易的论坛，有用户注册与登录、新建帖子、查看帖子、修改帖子、删除帖子、回复帖子等功能。前端使用 Bootstrap 框架，后端使用 PHP + MySQL。

## 当前目录结构

```
├── assets/             # 静态资源
├── controller/         # 后端业务
├── view/               # 前端视图
|
├── create.php          # 发表帖子
├── edit.php            # 编辑帖子
├── index.php           # 主页
├── login.php           # 用户登录页
├── posts.php           # 浏览某一个板块的所有帖子
├── register.php        # 用户注册页
├── user.php            # 个人中心页
├── view.php            # 查看某个帖子
|
├── docs/               # 技术文档
├── README.md           # 总览文档
```
